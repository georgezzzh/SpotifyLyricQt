## 安装
* Windows

支持Win10和Win11
在Release页面下载安装即可，windows安装之后桌面会显示SpotifyLyric图标，在电脑重启之后，会在菜单栏->Spotify Lyric找到本软件。Windows版本如果发现歌词无法同步，可以关闭网络代理或者重启系统。
* ubuntu下卸载本软件

ubuntu仅实测ubuntu20.04，其他版本受限于我系统的开发环境，暂无编译好的deb包。
`sudo dpkg -r spotifylyricQt`
